## Development

Install development tools:

- make sure you have `python3.7`, `python3.8`, `python3.9` aliases
- intall [poetry](https://python-poetry.org/docs/#installation)

Installing development dependencies:
```sh
poetry install
```

Runing tests:
```sh
poetry run tox -q
```
